# 全国ランキング設定手順

## 1. Googleスプレッドシートを作成

新しいGoogleスプレッドシートを作成します。シート名は自動で `rankings` が作成されるため、空のままで問題ありません。

## 2. Apps Scriptを開く

スプレッドシート上部メニューから **拡張機能 → Apps Script** を開きます。

## 3. GASコードを貼り付け

このZIP内の `gas/Code.gs` の内容をApps Scriptの `Code.gs` に貼り付けます。

## 4. 初期設定関数を実行

Apps Script上部の関数選択で `setupRankingSheet` を選び、実行します。初回は権限承認が必要です。

## 5. ウェブアプリとしてデプロイ

**デプロイ → 新しいデプロイ → 種類を選択 → ウェブアプリ** を選択します。

- 実行するユーザー: 自分
- アクセスできるユーザー: 全員

デプロイ後に表示される `/exec` で終わるURLをコピーします。

## 6. app.jsにURLを設定

`app.js` の下記行に、コピーしたGAS URLを貼り付けます。

```js
const RANKING_API_URL = "https://script.google.com/macros/s/xxxxxxxxxxxxxxxx/exec";
```

その後、GitHub Pagesへアップロードし直してください。

## 補足

- ランキングはジャンル別に保存されます。
- 登録される項目は、日時、名前、ジャンル、点数、正答数、平均回答時間、合計時間、アプリバージョンです。
- シンプルなJSONP方式のため、GitHub Pagesからでもランキング取得・登録できます。
- 不正送信を完全に防ぐ仕組みではありません。社内・身内向けや簡易ランキング向けです。
