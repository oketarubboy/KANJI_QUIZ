# 漢字よみがなテスト PWA v0.6.0

CSVで問題を管理できる、10問形式の漢字よみがなテストです。GitHub Pages にそのままアップロードして利用できます。

## 主な機能

- 1回10問出題
- 正答数と回答スピードで点数計算
- ジャンル選択
- CSV読み込み
- 端末内最高点保存
- 全国ランキング対応（Googleスプレッドシート + GAS）
- PWA対応
- バージョン表示と最新版更新ボタン

## 全国ランキング

`README_GAS_全国ランキング設定手順.md` の手順でGoogleスプレッドシートとGASを設定してください。GAS URLを `app.js` の `RANKING_API_URL` に入れると有効になります。
